# Excel Sales Dashboard & Data Analytics Project

Hi! This repository contains my full Excel data analytics work. I started with dynamic formulas and built it up to an interactive Executive Sales Dashboard across multiple sheets.

---

## 🚀 Project Architecture (All 5 Sheets Included)

To explore the live formulas, interactive charts, and dynamic slicers, please download **`Excel_Practice (1).xlsx`**. Inside the workbook, you will find these 5 specific sheets:

1. 📊 **`DASHBOARD`**: The main presentation view featuring interactive Column and Pie charts with clean data labels for quick business insights.
2. 📝 **`PROJECT_EXCEL`**: The cleaned raw data sheet utilizing advanced conditional formatting and data bars to automatically highlight top performance metrics.
3. 📦 **`PRODUCT_SALES`**: Backend analytics sheet leveraging Pivot Tables to break down sales trends by product categories.
4. 🏙️ **`CITY_SALES`**: Geographical data breakdown sheet tracking revenue and volume across different regions and cities.
5. 👥 **`CUSTOMER_SALES`**: Customer-centric analysis sheet mapping buyer patterns and top-performing client segments.

---

## 🛠️ Advanced Excel Skills Demonstrated
* **Data Modeling:** Connected datasets across sheets using **XLOOKUP** and **INDEX-MATCH**.
* **Formulas:** Used **SUMIFS**, **COUNTIFS**, and logical operators for dynamic tracking.
* **Interactive UI:** Implemented **Pivot Charts**, **Slicers**, and **Timeline Filters** for seamless user navigation.
