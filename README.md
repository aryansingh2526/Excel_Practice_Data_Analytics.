# Excel Sales Dashboard & Data Analytics Project

Hi! This repository contains my Excel data analytics work. I started with basic functions and built it up to an interactive Executive Sales Dashboard.

---

## 🚀 Main Project: Sales Performance Dashboard
To check out the main project, please download **`Excel_Practice (1).xlsx`**. 
Inside the workbook, you will find these specific sheets:
* **`DASHBOARD`**: This is the main view with interactive charts (Bar, Column, and Pie) and data labels for quick insights.
* **`PROJECT_EXCEL`**: The clean raw data with conditional formatting and data bars to highlight top numbers.
* **`PRODUCT_SALES` / `CITY_SALES` / `CUSTOMER_SALES`**: The backend work where I used Pivot Tables to break down sales trends.
